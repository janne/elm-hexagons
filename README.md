Hexagons
========

Initial work on hex tiled game art, using the [Hexagon tiles](http://opengameart.org/content/hexagon-tiles-93x) by [Kenney.nl](http://kenney.nl).

Setup
-----

* `npm install -g elm`
* `elm package install`
* `elm reactor`
* http://localhost:8000/src/Main.elm
